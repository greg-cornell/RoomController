# LightSwitch

An old hobby project I used to learn to write a C application on an RPi and interact with peripherals.  The idea was to use an RPi with a encoded rotary switch, a temperature sensor, and a FET to have a product that would act as not only a dimmable light controller but a room temperature sensor.

This repository was originally created as a personal experiment and has since been re-imported into GitHub as part of a broader effort to learn how to manage, version, and maintain legacy code.

---

## 🧭 Project Purpose

This project serves two main purposes:

1. **Original Intent**
   - Learn how to write an embedded C application.
   - Learn how to interact with hardware peripherals.

2. **Current Intent (Learning Exercise)**
   - Practice importing and organizing legacy code into GitHub
   - Improve repository hygiene (README, structure, commits, etc.)
   - Build muscle memory for version control workflows
